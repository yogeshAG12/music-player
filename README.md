Music Player – HTML, CSS & JavaScript

A simple and responsive music player web app built using HTML, CSS, and JavaScript.
It includes features like playlist, search, shuffle, repeat, and light/dark mode — similar to a mini Spotify clone.

Live Demo :

Open the Music Player

Features :

 * Play, pause, next, and previous controls

 * Shuffle and repeat options

 * Light and dark theme toggle

 * Search songs by title or artist

 * Dynamic playlist with cover images

 * Works with local audio files

Tech Stack :

 * HTML5 – Structure

 * CSS3 – Styling and layout

 * JavaScript – Functionality and interactivity
